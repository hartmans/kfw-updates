Kerberos with Command Prompt

Kerberos for Windows can be run from the Command Prompt in Windows. The following commands are available. Click a command name to see the relevant UNIX man page.

  • KINIT: Log in to Kerberos and obtain tickets.
  • KLIST: List all of your valid tickets.
  • KDESTROY: Destroy all of your tickets.
  • MS2MIT: Import Kerberos tickets from your Windows Logon Session, if available.
