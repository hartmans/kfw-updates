Using Kerberos Menus

MIT Kerberos uses a ribbon menu. Most commands and options are available in the two tabs in the ribbon at the top of the main window. The Home tab contains command buttons. The Options tab contains checkboxes for controlling ticket and view options.

A few commands and options are controlled in the Get Ticket window. The Get Ticket window allows you to choose ticket settings and to clear your principal history.

Browse the tables below to see what each button and option does. Or click here for an introduction to using MIT Kerberos.

Home Tab

MIT Kerberos opens with the Home tab visible. Use the buttons in the Home tab to work with tickets and passwords and to change your default principal. Depending on your installation and your needs, you might not need every button.

The table below gives a quick explanation of each of the buttons. To jump to a detailed How To page for a button, click the button name.

Get Ticket Get a new Kerberos ticket.
Renew Ticket Renew tickets for the selected principal(s). Renewing a ticket extends its valid lifetime, resetting it to the length of the original ticket.
Destroy Ticket Destroy all tickets. Not available if the Automatic Import Tickets option is selected and you have imported tickets.
Make Default Make the selected principal the default principal. Not necessary if you have only one principal.
Export Ticket Export your tickets to your Windows Logon session for use with Windows services. This will destroy any tickets you already have in your Windows Logon session.
Import Tickets Import Windows domain tickets (obtained through a Windows Logon session) into MIT Kerberos. This will destroy any tickets you already have in MIT Kerberos.
Change Password Change your password.

Options Tab

Click the Options tab to reach the checkboxes that control ticket and view options. The Options tab has two panels. Use the View Options panel to choose which information is displayed for your tickets. Use the Ticket Options panel to turn MIT Kerberos's automatic features on or off.

View Options Panel

The View Options checkboxes control what ticket information is displayed. Select a checkbox to see the information column it controls, or deselect the checkbox to hide it.

The table below gives a quick explanation of each of the View options. To jump to a more detailed explanation of an option, click an option name.

Option Description
Issued The date and time that your tickets were issued.
Renewable Until The date and time when you will no longer be able to renew your current tickets.
Valid UntilWhen your ticket will expire.
Encryption Type The type of encryption used to encrypt your tickets and session keys.
Flags How the tickets were flagged (renewable and/or fowardable) when you obtained them.
Import Status This column is available when you have obtained a Kerberos ticket by logging into a Windows domain with a Windows Logon session. It shows whether those tickets have been or can be imported, and whether tickets obtained with the Get Ticket window are exportable or exported.

Ticket Options Panel

The Ticket Options checkboxes control MIT Kerberos's automatic features. Select a checkbox to turn a feature on, or deselect a checkbox to turn the feature off.

The table below gives a quick explanation of each of the Ticket options. To jump to a more detailed explanation of an option, click an option name.

Option Description
Destroy Tickets on Exit Have MIT Kerberos destroy your tickets when you exit the program.
Automatic Ticket Renewal Automatically renew your Kerberos tickets for their entire renewable lifetime, without promptings or requiring a password, when possible.
Expiration Alarm Have MIT Kerberos provide an audible alarm 15, 10, and 5 minutes before your tickets expire.
Allow Mixed Case Realm Name Allow you to get tickets for a realm that has a name that includes lower case letters.
