Using Kerberos Menus

Kerberos for Windows uses a ribbon menu. All commands and options are available in the ribbon of buttons and checkboxes at the top of the main window.

The ribbon menu has four panels. Left to right, there are buttons, the View checkboxes, the Options checkboxes, and the More panel. If your Kerberos window is not wide enough to display everything, the Options and More panels shrink to pull-down menus.

Browse the tables below to see what each button and option does. Click a panel, command, or option name to open more detailed help for that subject. Or click here for introduction to using Kerberos for Windows.

Buttons
Use these buttons to work with tickets and passwords. Click a button name to reach detailed help for using that feature. Using the Button Panel
Get Ticket Get a new Kerberos ticket.
Renew Ticket Renew tickets for the selected principal(s). Renewing a ticket extends its valid lifetime, resetting it to the length of the original ticket.
Destroy Ticket Destroy all tickets. Not available if you have imported tickets and the Automatic Import option is selected.
Export Ticket Export ticket to your Windows Logon session for use with your Windows domain.
Make Default Make the selected principal the default principal. Not necessary if you have only one principal.
Change Password Change your password.

View Panel
Use these checkboxes to choose what information is displayed for your tickets. Using the View Panel
Issued The date and time that your tickets were issued.
Renewable Until The date and time when you will no longer be able to renew your current tickets.
Valid UntilWhen your ticket will expire.
Encryption Type The type of encryption used to encrypt your tickets and session keys.
Flags How the tickets were flagged (renewable and/or fowardable) when you obtained them.
Import Status This column is available when you have a Kerberos ticket obtained by logging into a Windows domain with a Windows Logon session. It shows whether those tickets have been or can be imported, and whether tickets obtained with the Get Ticket window are exportable or exported.

Options Panel
Use these checkboxes to change how ticket renewals and expirations are handled. Using the Options Panel
Destroy Tickets on Exit Have Kerberos for Windows destroy your tickets when you exit the program. Click for details
Automatic Ticket Renewal Automatically renew your Kerberos tickets for their entire renewable lifetime, without promptings or requiring a password, when possible. Click for details
Expiration Alarm Have Kerberos provide an audible alarm 15, 10, and 5 minutes before your tickets expire. Click for details

More Panel
Use these buttons to access features not needed by all users.
Import Ticket Import the ticket obtained for a Windows domain when you logged into a Windows Logon session.
Forget Principals Delete all saved principals from the auto-complete list in the Get Ticket and Change Password windows.
Allow Mixed Case Realm Name Allows you to get tickets for a realm that has a name that includes lower case letters. Click for details
