Change Password

To change your password:

  1. Create or choose a strong new password. It's best to start with a phrase instead of a word.
    Password tips and examples
  2. If you have multiple principals, you can select the principal you want to manage by clicking it in the main window. Or you can wait to choose your principal in the Change Password window.
  3. Click the Change Password button in the ribbon menu at the of the main window.
  4. Verify your principal (your Kerberos identity) or select a different one with the pull down arrow.
  5. Enter your current password.
  6. Enter your new password twice. You must enter it the second time to make sure you have typed it correctly, since you cannot visually confirm what you have typed.
  7. Click Okay.

Remember to use your new password the next time you obtain tickets.

Imported Ticket Note: If you change the password for a principal with imported tickets, you must use your new password when you use that principal to log on to your Windows domain with a Windows Logon session. About importable (Windows domain) tickets
