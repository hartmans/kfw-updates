Reporting Bugs and Requesting Assistance

We strive to provide a robust product. If you find bugs in Leash or Keberos, we want to know about them. Please email bug reports to kfw-bugs@MIT.EDU. Provide as much detail as you can so that we can replicate the issue.

There is a Usenet newsgroup dedicated to discussing Kerberos issues: comp.protocols.kerberos. The mailing list kerberos@MIT.EDU is gatewayed to the newsgroup.

To subscribe to the mailing list, email a request to kerberos-request@MIT.EDU or submit the subscribtion form on the web page
