Encryption Types

Kerberos supports several types of encryption for the session key and the ticket. The type used for a particular ticket or session key depends on a combination of your operating system, the configuration of your Kerberos installation, and the configuration and capabilities of the particular network resources you are accessing.

Some encryption types that are still supported are now considered weak and therefore undesirable. However, they are still supported to allow Kerberos to be used in a network that contains some older machines running operating systems that do not support the newer encryption types.

View Encryption Types

In the View section of the menu, select Encryption Type to open that column. The column will show the encryption type associated with each of your tickets and session keys.

Supported Encryption Types

Encryption Type Description
des- The DES (Data Encryption Standard) family is a symmetric block cipher. It was designed to handle only 56-bit keys which is not enough for modern computing power. It is now considered to be weak encryption.
  • des-cbc-crc (weak)
  • des-cbc-md5 (weak)
  • des-cbc-md4 (weak)
  • des-cbc-raw (weak)
  • des-hmac-sha1(weak)
des3- The triple DES family improves on the original DES (Data Encryption Standard) by using 3 separate 56-bit keys. Some modes of 3DES are considered weak while others are strong (if slow).
  • des3-cbc-sha1
  • des3-cbc-raw (weak)
  • des3-cbc-sha1
  • des3-hmac-sha1
  • des3-cbc-sha1-kd
aes The AES Advanced Encryption Standard family, like DES and 3DES, is a symmetric block cipher and was designed to replace them. It can use multiple key sizes. Kerberos specifies use for 256-bit and 128-bit keys.
  • aes256-cts-hmac-sha1-96
  • aes128-cts-hmac-sha1-96
rc4 or
arcfour		 The RC4 (Rivest Cipher) is a symmetric stream cipher that can use multiple key sizes. The exportable variations are considered weak, but the others are strong.
  • arcfour-hmac
  • rc4-hmac
  • arcfour-hmac-md5
  • arcfour-hmac-exp (weak)
  • rc4-hmac-exp (weak)
  • arcfour-hmac-md5-exp(weak)
