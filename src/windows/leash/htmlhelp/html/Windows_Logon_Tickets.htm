Windows Domain Tickets

MIT Kerberos is not the only interface for managing Kerberos tickets. When you log on to a Windows domain, you are issued a Kerberos ticket through your Windows Logon session. This ticket is automatically renewed until you log out of the session, when it is destroyed.

Sometimes applications that require Kerberos authentication only work with MIT Keberos. Others work only with the interface that is part of the Windows Logon session. For this reason, you can use MIT Kerberos to either import tickets from your Windows Logon session or to export tickets into it, depending on your needs.
