Import Tickets

If your Windows Logon Session uses Kerberos for authentication, you can import your existing ticket(s) into Leash instead of getting a fresh ticket.

Note: Importing tickets will destroy any tickets you already have in Leash. If you have unexpired tickets when you run the Import Ticket command, Leash will warn you and give you the option to cancel the command.

To import tickets from your Windows Logon Session:

  1. Click the Import Ticket button in the Home tab.
  2. Click Okay to confirm that you want to import the tickets and destroy any you already have in Leash.

You can now view and manage your Kerberos ticket with Leash.
